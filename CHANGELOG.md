# Changelog

## v0.1.0

v0.1.0 is a portfolio-ready MVP release package for QuoteOps AI. It packages the completed deterministic pricing operations workflow, demo support, deployment preparation, and verification guardrails. It should not be presented as production-ready or autonomous.

### Included MVP Areas

- Backend foundation
- Quote preview
- Candidate price generation
- Price validation
- Human approval/rejection workflow
- Safe explanations
- Audit logs
- CSV import/export
- Pricing simulations
- Customer quote requests
- Price table history/comparison
- Workflow jobs
- Strategy templates
- KPI dashboard
- Dashboard insights
- Scenario comparisons
- HTML reports
- Demo data tools
- PostgreSQL configuration hardening
- Render backend/frontend deployment preparation
- CORS/env cleanup
- Health/readiness/status hardening
- Frontend navigation cleanup
- Error/loading/empty state UX
- Docs consistency cleanup
- Security checks
- Final regression tests
- Render deployed QA support
- Portfolio README polish
- Demo flow polish

### Safety Boundaries

- No automatic price approval
- No automatic price table activation
- No real competitor scraping
- No email sending
- No payment flow
- No external AI required by default
- Human review required before business use

### Release Boundary

This PR-34 package prepares the release materials only. The actual git tag and GitHub Release are planned for PR-35 after final verification.
