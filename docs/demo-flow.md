# Demo Flow

Use this flow for a portfolio walkthrough or local MVP demo.

1. Check backend health and system status.
2. Sign in with a local demo user.
3. Seed demo data if needed.
4. Review product and cost profile data.
5. Generate a quote preview.
6. Generate candidate prices.
7. Run price validation.
8. Submit an approval request.
9. Approve or reject through the human workflow.
10. Review audit logs.
11. Run a pricing simulation.
12. Create a scenario comparison.
13. View KPI dashboard metrics.
14. View dashboard insights.
15. Generate an HTML report.

## Safety Boundaries During Demo

- No AI-generated price approval is performed.
- No price table is automatically activated.
- No email is sent.
- No external scraping is performed.
- Demo reset tools are local/demo utilities and are not a production reset workflow.
