# QuoteOps AI v0.1.0

Release type: Portfolio-ready MVP release package

QuoteOps AI v0.1.0 packages the deterministic pricing operations MVP with quote preview, candidate pricing, validation, human approval workflow, audit logs, simulations, dashboard metrics, reports, demo tooling, deployment preparation, and regression/security checks.

## What Is Included

- Core pricing operations
- Human approval workflow
- Dashboard and insights
- Reports
- Demo tools
- Deployment preparation
- Security and regression checks
- Documentation

## What Is Not Included

- No automatic price activation
- No automatic price approval
- No email sending
- No payment flow
- No real competitor scraping
- No production migration system
- No GitHub release tag yet in PR-34

## Safety Boundaries

QuoteOps AI v0.1.0 requires human review before business use. Numeric pricing artifacts are deterministic backend outputs, and the system does not activate price tables, send customer quotes, scrape competitors, process payments, or send email as part of this release package.
