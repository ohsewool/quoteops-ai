# Demo Presenter Script

Use this as a short spoken script for a school, project, or portfolio presentation.

## Opening

"QuoteOps AI is a deterministic pricing-operations SaaS MVP for small print and custom-product businesses. It helps an operator move from a customer quote request to pricing review, validation, approval workflow, audit logs, dashboards, and review reports."

## Problem Statement

"Many teams still handle quote pricing with spreadsheets, manual checks, and unclear approval steps. That can make prices inconsistent, risky, and hard to audit later."

## Solution Overview

"QuoteOps AI centralizes the workflow in one web app. The backend performs deterministic calculations for quote preview, candidate prices, validation, simulations, and scenario comparison. The UI gives operators and managers a guided way to inspect results, approve or reject through a human workflow, and review audit history."

## Main Demo Flow

"I will start with health and system status, then use demo data to inspect products and cost profiles. Next I will show a customer quote request, generate a quote preview, generate candidate prices, run validation, and submit the result for human approval. After that I will show audit logs, pricing simulation, scenario comparison, KPI dashboard, dashboard insights, and an HTML report."

## Technical Highlights

"The project uses FastAPI, SQLite locally, PostgreSQL-ready configuration, React/Vite, role-aware API access, audit logging, OpenAPI-visible routes, regression tests, security checks, and Render deployment preparation. Pricing numbers are calculated in backend services rather than guessed by an AI model."

## Safety Boundaries

"This demo does not perform AI-generated price approval. It does not automatically activate a price table, send a quote to a real customer, scrape competitors, process payments, or send email. Human review is required before any business use."

## Closing

"The value of the project is the complete operating workflow: deterministic pricing artifacts, validation, human decision points, auditability, and a demo-ready SaaS interface. It is a portfolio MVP, not a production pricing engine."
